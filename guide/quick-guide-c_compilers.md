
## Компиляторы
### GNU Compiler Collection
[gcc](https://gcc.gnu.org/)
```sh
apt-get install gcc

gcc --version
```
#### Компиляция и запуск
```sh
cc hello.c
./a.out

# Или так
cc -o hello hello.c
./hello
```


### Clang
[clang](https://clang.llvm.org/)
```sh
apt-get install clang

clang --version
```
